堆模板

发表于 2023-11-23

首页

利用基于数组的完全二叉树实现。

如果数组有效下标从1开始，则下标idx的父节点为idx // 2，左孩子结点为idx * 2，右孩子节点为idx * 2 + 1。

如果数组有效下标从0开始，则下标idx的父节点为(idx - 1) // 2，左孩子结点为idx * 2 + 1，右孩子节点为idx * 2 + 2。
